README FILE

1. CSS Types Used:
• Inline CSS – Used for background color and padding of the outer <div>
• Internal CSS – Used for headings, bio, contact section, and links
• External CSS – Main styles like card layout, image styling, borders

2. CSS Selectors Used:
• Element Selectors – h1, h2, ul, li
• Class Selectors – .card, .bio, .contact
• ID Selector – #profile-pic
• Group Selector – h1, h2
• Descendant Selector – .contact p
• Attribute Selector – a[href]

3. Features Added:
• Centered card using `margin: auto`
• White background, solid black border, rounded corners
• Circular profile image
• Hover effect on links
